# Maven+SpringMVC+Spring+Mybatis+MySql消费查询系统


# 一、设计背景
目前很多商场为了刺激消费，纷纷推出了VIP会员服务。顾客在商场消费满一定金额后，顾客可申请为VIP，商家记录VIP的每次消费，累积消费金额，提升VIP等级，不同的VIP等级享有不同的优惠折扣
本系统是一个内部系统，仅供商家工作人员使用，主要功能有：

VIP用户录入

VIP用户查询

VIP用户管理

VIP消费记录管理

商品管理
